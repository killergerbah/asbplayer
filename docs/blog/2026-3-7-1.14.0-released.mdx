Extension v1.14.0 was released a couple weeks ago with very little involvement from me.

<!-- truncate -->

The flagship feature of this release is subtitle annotations, driven entirely by [ShanaryS](https://github.com/ShanaryS).
ShanaryS will continue helping out with the project as a maintainer, building  on top of their impressive work.

I would also like to acknowledge new contributors:
- [@iamllama](https://github.com/iamllama) added hover tooltips for keyboard shortcuts.
- [@Hawkheart](https://github.com/Hawkheart) fixed up integration the Jelly integration. 
- [@danthemango](https://github.com/danthemango) added subtitle detection for iwanttfc.com.
- [@L-M-Sherlock](https://github.com/L-M-Sherlock) improved "jump to subtitle" UX, and fixed up the one-click Yomitan integration.
- [@Hit2Skill](https://github.com/Hit2Skill) enabled sidebar support on Firefox

Thank you!
